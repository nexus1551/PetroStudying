**Сегодня мы изучем как поставим свой fail2ban на сервер debian**
-------------------------------------------------------------
1) Обновим сервер до актуального <kbd>#</kbd>apt update && apt upgrade -y<kbd>#</kbd>
2) Установим fail2ban на сервер  <kbd>#</kbd>apt install fail2ban -y<kbd>#</kbd>
3) Основаная директория будет находиться в /etc/fail2ban/


    - Так как основной файл jail.conf лучше не трогать, создадим jail.local и настроем его полностью



--------------------------------------------------------------
**/etc/fail2ban/jail.local**
- [sshd]
- enabled = true   #включаем его
- port = ssh   #выбираем порт
- logpath = /var/log/auth.log   #где будут находится все логи
- maxretry = 3   #максимальное кол-во попыток
- bantime = 7200   #время бана в секундах
- bantime.increment = true   #увеличение бана при неправильном вводе
- bantime.factor = 2   #во сколько раз увеличится бан
--------------------------------------------------------------


На свой сервер мне надо было настроить только fail2ban на ssh из-за этого других настроек тут нет
 
 -------------------------------------------------------------- 
4) После всей настройки перезагружаем fail2ban <kbd>#</kbd>systemctl restart fail2ban<kbd>#</kbd>

  ------------------------------------------------------------
  
  **Проверить работает или нет можно по командам**
  
  - <kbd>#</kbd>systemctl status fail2ban<kbd>#</kbd>    #Просто проверяем работает ли сама служба
  - <kbd>#</kbd>fail2ban-client status sshd<kbd>#</kbd>    #Проверяем забанненых пользователей по ssh
